# WSFMDB
本地数据库存储 NSObject 类目  （灵感来自于 leancloud）
基于 runtime，类目，FMDB  在类初始化的方法中 创建模型的在数据库中对应表， 目前还有同步方法 ，后期会继续更新 ， 做到更好
集成方便使用简单。
